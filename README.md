# 📊 Enterprise KPI Management & Employee Performance Dashboard

A comprehensive, data-driven HR analytics and performance evaluation dashboard built with React. This application provides enterprise-grade tools to manage departmental KPIs, calculate employee bonus pools based on performance thresholds, and model company-wide profit factor metrics in real-time[cite: 3].

🎨 **Design & Development by:** Mehdi Salman Manesh
📌 **Live Demo:** [bravo-gauge-spark.lovable.app](https://bravo-gauge-spark.lovable.app/)

---

## 🚀 Key Features & Architectural Modules

The platform delivers deep organizational insights through a highly interactive and modular interface[cite: 3]:

*   **📈 Advanced KPI & Threshold Fine-Tuning:** Dynamic engines to update KPI values, weights, and set granular thresholds (Target, Stretch, and Minimum performance metrics) per employee[cite: 3].
*   **💸 Automated Bonus Pool Calculations:** Computes individual and department-wide bonuses dynamically tied to corporate net profits, target bonuses, and unique business development incentives[cite: 3].
*   **👥 Dynamic Workforce Management:** Allows HR managers to instantly add or remove employees, update names/roles, and customize department-specific multipliers[cite: 3].
*   **📊 Cross-Department Comparison Charts:** Utilizes visual comparison modules to analyze multi-departmental efficiency, average performance trends, and headcount data at a glance[cite: 3].
*   **🌐 Localization & Seamless Multi-Language Layout:** Built with a clean localization engine (i18n) via React Context API to serve multi-regional corporate structures[cite: 3].

---

## 🛠️ Tech Stack & Production Standards

*   **Core Architecture:** React.js powered by TypeScript for reliable, type-safe data structures[cite: 3].
*   **Styling & UI Components:** Tailwind CSS utilizing high-fidelity custom design components (glassmorphism layouts and Lucide icon kits)[cite: 3].
*   **State Management:** React Context API and specialized custom hooks (`useKPIData`) ensuring instant calculation updates across the DOM without delays[cite: 3].
*   **Build Utility:** Vite for optimized production bundling and instant Hot Module Replacement (HMR).
